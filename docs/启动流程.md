# 启动流程

## 一、集群

```bash
ssh root@192.168.157.121
# 密码 123456
start-all.sh
```

## 二、Flink JAR

1. IDEA 打开 `flink/` → Maven → `clean` → `package`
2. 浏览器打开 http://192.168.157.121:8081
3. Submit New Job → 上传 `flink/target/delivery-flink-poc-1.0-SNAPSHOT.jar` → Submit

## 三、本地服务（开三个终端）

```bash
# 终端 1：模拟器
cd generator && uv run python simulator/run.py

# 终端 2：FastAPI
cd server && uv run uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload

# 终端 3：前端
cd web && pnpm dev
```
