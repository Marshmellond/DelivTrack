# DelivTrack 启动手册

## 一、集群启动（SSH 到 Node_01）

```bash
ssh root@192.168.157.121
# 密码: 123456

start-all.sh
```

等待所有服务 OK，确认 Flink WebUI：

> http://192.168.157.121:8081

## 二、Flink JAR 提交

1. IDEA 打开 `flink/` 项目
2. Maven → clean → package
3. 打开 http://192.168.157.121:8081
4. Submit New Job → 上传 `flink/target/delivery-flink-poc-1.0-SNAPSHOT.jar` → Submit

## 三、种子数据（首次运行）

```bash
cd server
uv run python seed_full_data.py
```

## 四、本地服务（开 3 个终端）

```bash
# 终端 1 — 数据模拟器
cd generator
uv run python simulator/run.py

# 终端 2 — FastAPI
cd server
uv run uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload

# 终端 3 — Next.js 前端
cd web
pnpm dev
```

## 五、打开页面

> http://localhost:3000

管理员账号：`user0001` / `123456`

## 快速重置命令

```bash
# 清空所有聚合数据 + 重建 Kafka Topic
ssh root@192.168.157.121 clean-data.sh

# 重新生成种子数据
cd server && uv run python seed_full_data.py

# 停止集群
ssh root@192.168.157.121 stop-all.sh
```
